📊 性能监控功能

性能监控已自动开始收集数据

SDK 会自动监控以下性能指标：

  • 页面加载时间 (Navigation Timing)
  • 首次绘制 (FP) 和 首次内容绘制 (FCP)
  • 最大内容绘制 (LCP)
  • 首次输入延迟 (FID)
  • 累积布局偏移 (CLS)
  • FPS (帧率)
  • XHR 和 Fetch 请求
  • 首屏加载时间
6. XHR 网络请求

7. Fetch 网络请求

13. 手动上报自定义数据
